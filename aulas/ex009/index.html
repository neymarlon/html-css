lista de definições

HTML
liguagem de marcação para criação de conteúdo de um site, não é considerada uma linguagem de programação, portanto, é errado se dizer "eu programo em HTML"
CSS
linguagem de marcação para criação do design de um site, que também funciona na mesma regra do HTML, onde não é considerada uma linguagem de programação
JavaScript
linguagem de programação para a criação de interatividade de um site, que fecha a tríade inicial da criação, visto que os três funcionam muito bem juntos
